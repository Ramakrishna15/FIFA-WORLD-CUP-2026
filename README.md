# 🏆 FIFA World Cup 2026 — Fan App

A fully static, no-framework web app covering the FIFA World Cup 2026 (USA · Canada · Mexico).

## Features

- **Home** — Hero section with tournament stats and Golden Ball winners (Messi 2022/2014, Modrić 2018) with real photos
- **Teams** — All 48 teams clustered by Groups A–L with real flag images
- **Players** — Star player cards with Wikipedia photos, stats, and full profile modals
- **Groups** — Group stage overview for all 12 groups
- **Schedule** — All 72 group stage matches + knockout fixtures with real June 2026 results
- **Bracket Simulator** — Pick winners or simulate the entire knockout bracket randomly
- **Match Predictor** — Head-to-head predictor using team strength ratings and squad data

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no build tools
- Wikipedia API for player photos
- flagcdn.com for country flag images
- Wikimedia Commons for POTM ceremony photos

## Getting Started

Just open `index.html` in any browser — no server or install needed.

```bash
git clone https://github.com/Ramakrishna15/WC2026.git
cd WC2026
# open index.html in your browser
```
